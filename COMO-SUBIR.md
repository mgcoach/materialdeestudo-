## Passo a passo
1. Descompacte este ZIP no celular (app Arquivos do iPhone abre zip).
2. No GitHub, repositório `materialdeestudo-`, "Add file" > "Upload files".
3. Selecione TODOS os arquivos .html + o estudos.json de uma vez.
4. Arquivos de mesmo nome são substituídos automaticamente.
5. "Commit changes", aguarde 2-3 min, teste numa aba anônima.

## IMPORTANTE
