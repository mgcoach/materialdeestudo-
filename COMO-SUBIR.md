# Como atualizar o site MG COACH (todos os arquivos de uma vez)

Este ZIP contém o site completo com o novo tema claro/escuro já embutido.

## Passo a passo (GitHub no celular ou PC)

1. Abra o repositório `materialdeestudo-` no GitHub.
2. Toque em **"Add file"** → **"Upload files"**.
3. Selecione **TODOS os arquivos** deste ZIP de uma vez:
   - 14 arquivos .html (incluindo o index.html)
   - estudos.json
4. Os arquivos com o mesmo nome serão **substituídos automaticamente**.
5. Role até o fim e toque em **"Commit changes"**.
6. Aguarde 2-3 minutos para o GitHub Pages atualizar.
7. Abra o site numa **aba anônima** (para evitar cache) e teste o botão de tema.

## Importante
- NÃO descompacte e suba a pasta; suba os ARQUIVOS (o GitHub Pages precisa do index.html na raiz).
- O tema fica salvo na escolha de cada visitante (claro ou escuro).
- O padrão continua sendo o escuro (identidade MG COACH).
